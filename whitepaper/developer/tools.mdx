## CLI Tool

The VEXARA CLI provides command-line access to all platform features.

### Installation

```bash path=null start=null
npm install -g @vexara/cli
```

### Authentication

```bash path=null start=null
vexara auth login
```

### Commands

**Track a wallet**
```bash path=null start=null
vexara wallet track 7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU
```

**Get transaction history**
```bash path=null start=null
vexara wallet txs 7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU --limit 20
```

**Export data to CSV**
```bash path=null start=null
vexara export --wallet ADDRESS --format csv --output data.csv
```

## Webhook Integration

Receive real-time notifications via HTTP callbacks.

### Setup

Configure webhooks in your dashboard at `app.vexara.io/webhooks`.

### Event Types

| Event | Description | Payload |
|-------|-------------|---------|
| `wallet.transaction` | New transaction detected | Transaction object |
| `wallet.balance` | Balance changed | Balance data |
| `wallet.token` | Token account updated | Token info |
| `alert.triggered` | Custom alert fired | Alert details |

### Webhook Payload

```json path=null start=null
{
  "event": "wallet.transaction",
  "timestamp": 1728936066,
  "data": {
    "address": "7xKXtg2CW87d97TXJSDpbD5jBkheTqA83TZRuJosgAsU",
    "signature": "3xZR8...",
    "type": "transfer",
    "amount": 10.5,
    "from": "...",
    "to": "...",
    "blockTime": 1728936000
  }
}
```

### Security

All webhook requests include a signature header for verification:

```typescript path=null start=null
import crypto from 'crypto';

function verifyWebhook(payload: string, signature: string, secret: string): boolean {
  const hmac = crypto.createHmac('sha256', secret);
  const digest = hmac.update(payload).digest('hex');
  return crypto.timingSafeEqual(
    Buffer.from(signature),
    Buffer.from(digest)
  );
}
```

## GraphQL Playground

Interactive GraphQL API explorer available at `api.vexara.io/graphql`.

### Example Query

```graphql path=null start=null
query GetWallet($address: String!) {
  wallet(address: $address) {
    address
    balance
    transactions(limit: 10) {
      signature
      type
      amount
      blockTime
    }
    tokens {
      mint
      balance
      symbol
    }
  }
}
```
